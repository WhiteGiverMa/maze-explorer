# 迷宫探索者 (Maze Explorer)

Godot 4 制作的俯视角迷宫探索游戏。

## 三种模式

| 模式 | 时间 | 地图 | 特色 |
|------|------|------|------|
| ⚡ 速通 | 10分钟 | 80×80 | 纯迷宫探索 |
| 🎯 标准 | 30分钟 | 140×140 | 迷宫 + 道具 |
| 🏰 深度 | 60分钟 | 200×200 | 迷宫 + 道具 + 怪物(开发中) |

## 操作

| 按键 | 功能 |
|------|------|
| WASD/方向键 | 移动 |
| 左键小地图 | 放大全览 |
| Esc | 暂停菜单 |
| Z | 作弊面板 |
| V | 游戏说明 |
| 1-5 | 使用道具 |

## 作弊码

`Z` 键打开面板，`Enter` 执行：`xieyihui` `chuanqiang` `shunyi` `shubiao` `shengli` `exit` `time N`

## 运行

需要 [Godot 4.7 Mono](https://godotengine.org/download/) + [.NET 8.0 SDK](https://dotnet.microsoft.com/download/)
